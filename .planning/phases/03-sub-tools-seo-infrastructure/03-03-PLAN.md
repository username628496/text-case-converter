IMPORTANT: Before writing, read the Next.js 16 sitemap docs in `node_modules/next/dist/docs/` to confirm the exact API signature. The `alternates.languages` format may differ from Next.js 14/15. Verify and use the installed version's API.

```typescript
import type { MetadataRoute } from 'next'
import { tools } from '@/lib/tools'

const BASE_URL = 'https://textcaseconverter.com'

export default function sitemap(): MetadataRoute.Sitemap {
  return tools.map((tool) => {
    const enUrl = tool.isHomepage
      ? `${BASE_URL}/`
      : `${BASE_URL}/${tool.slug}/`
    const viUrl = tool.isHomepage
      ? `${BASE_URL}/vi/`
      : `${BASE_URL}/vi/${tool.slug}/`

    return {
      url: enUrl,
      lastModified: new Date(),
      alternates: {
        languages: {
          en: enUrl,
          vi: viUrl,
        },
      },
    }
  })
}
```

This produces 5 entries (one per tool) each with EN + VI alternates. The `url` field uses the EN URL as primary. `alternates.languages` provides hreflang entries for both locales.
